## 🚀 Установка

### Требования
- Python 3.10 или выше
- pip (менеджер пакетов Python)

### Шаги установки

1. **Клонируйте репозиторий:**

```bash
git clone https://github.com/your-username/ai-chat-client.git
cd ai-chat-client
````

2. **Установите зависимости:**

```bash
pip install -r requirements.txt
```

3. **Получите бесплатный API ключ:**

* Перейдите на [openrouter.ai/keys](https://openrouter.ai/keys)
* Зарегистрируйтесь (бесплатно)
* Создайте новый API ключ

4. **Запустите приложение:**

```bash
python main.py
```

## 📖 Использование

### Первый запуск

1. Нажмите ⚙️ **Настройки** в боковой панели.
2. Вставьте ваш API ключ OpenRouter.
3. Выберите модель AI (рекомендуется Mistral 7B).
4. Нажмите 💾 **Сохранить**.

### Работа с чатами

| Действие            | Как сделать             |
| ------------------- | ----------------------- |
| Новый чат           | Кнопка ✨ Новый чат      |
| Отправить сообщение | Enter или кнопка ➤      |
| Новая строка        | Shift + Enter           |
| Удалить чат         | Кнопка ✕ справа от чата |
| Экспорт             | 📥 Экспорт чата внизу   |

### Настройка промптов

В настройках можно задать системный промпт, например:

* Ты — опытный программист. Отвечай с примерами кода.
* Ты — учитель английского. Исправляй ошибки в моих текстах.
* Отвечай кратко и по существу.

## ❓ FAQ

<details> <summary><b>Как получить API ключ бесплатно?</b></summary>
Зайдите на openrouter.ai  
Нажмите "Sign In" и зарегистрируйтесь  
Перейдите в Keys  
Нажмите "Create Key"  
Скопируйте ключ (начинается с sk-or-)
</details>

<details> <summary><b>Почему модель не отвечает?</b></summary>
Проверьте подключение к интернету  
Убедитесь, что API ключ введён правильно  
Попробуйте другую модель (Mistral 7B самая стабильная)  
Подождите минуту (может быть rate limit)
</details>

<details> <summary><b>Где хранятся мои чаты?</b></summary>
Все данные хранятся локально в папке data/:
- `chats.db` — база данных с историей
- `settings.json` — ваши настройки
</details>

<details> <summary><b>Это бесплатно?</b></summary>
Да! Приложение полностью бесплатное. OpenRouter предоставляет бесплатный доступ к некоторым моделям с ограничением по количеству запросов.
</details>
